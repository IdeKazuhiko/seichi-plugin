■概要
　spigot1.18.1上で動作する整地補助のプラグインです。
　ある程度動作確認はしていますが、ご利用の際は自己責任でお願いします。

■プラグインの機能
　1.木の棒を持っている状態でブロックを壊した時、プレイヤーの直進方向に10マス、左右方向に５マス、上方向に５マスブロックを自動破壊します。
　2.松明を持っている状態でブロックを壊した時、下向きの円錐状にブロックを自動破壊します。
　3.任意をブロックを設置した時、10×10マス分同じブロックを設置します。

■使い方
　1.spigotのディレクトリ/plugins/配下にseichi_plugin-0.1.jarを配置します。
　2.サーバーを再起動、または/reloadコマンドでブラグインを再読み込みします。
　3.使用をやめたい場合は、/seichi offコマンドでプラグインの処理が停止します。
　　再度使用したい場合は、/seichi onコマンドを入力してください。

■開発環境
